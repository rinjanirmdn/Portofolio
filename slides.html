Thesis Project

Warehouse App + QR Code

Academic thesis at PT Globalnine Indonesia, 2024 - 2025

A web-based warehouse management app that uses QR Code labels for real-time inventory tracking. Each QR code is generated with Reed-Solomon error correction and validated through SHA-256 hashing to prevent tampering and cloning.

Reed-Solomon Error Correction

Generates high-durability QR code labels that remain scannable even with partial damage or print defects.

SHA-256 Data Integrity

Every scan is validated against a SHA-256 hash to confirm authenticity and prevent barcode cloning.

Inventory Tracking Dashboard

Warehouse staff can view real-time stock levels, item locations, and transaction history through a clean web interface.

Scanning System Thesis

This project reduced manual administration time and errors in cargo records at the warehouse. It was developed, tested, and validated as part of my undergraduate thesis at Universitas Bhayangkara Jakarta Raya.
